· 2019-06-18 ·

NODE 及 VUE 环境搭建

NODE 下载地址

安装成功后 node -v 查看版本

更换镜像为淘宝源

npm config set registry https://registry.npm.taobao.org

使用 cnpm

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装vue脚手架

cnpm install -g vue-cli

创建VUE项目

vue init webpack 项目名称

VUE 项目启动

npm run dev

VUE 项目打包

npm run build

项目部署

找到 项目文件夹中的 dist 放到tomcat webapp中即可
